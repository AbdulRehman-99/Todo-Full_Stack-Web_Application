# Frontend Overview

## Purpose
This specification defines the **frontend layer** of the Todo App.  
It focuses on **user interface, task CRUD interactions, and visual components**. The frontend communicates with backend APIs to fetch and update tasks.

## Scope
- Display all tasks for the logged-in user
- Allow users to:
  - Add new tasks
  - Update task title/description
  - Delete tasks
  - Mark tasks as complete/incomplete
- Map API responses to UI components
- Apply consistent styling with Tailwind CSS
- Ensure responsive layout for desktop and mobile

## Technology Stack
- Next.js 16+ (App Router)
- TypeScript
- Tailwind CSS
