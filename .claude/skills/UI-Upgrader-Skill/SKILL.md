UI Skills – Next.js Frontend Enhancement

Skill 1: Layout & Visual Hierarchy
- Improve spacing, alignment, and sectioning
- Use containers, cards, and consistent padding
- Establish clear visual hierarchy (heading → content → actions)
- Apply modern layout patterns (centered content, max-widths)

Skill 2: Interactive States & Feedback
- Add hover, focus, active, and disabled states
- Improve button, input, and list item feedback
- Add empty states, loading states, and subtle UI hints
- Use Tailwind transitions for smooth UX

Skill 3: Task UI Polish
- Improve TaskList and TaskItem presentation
- Use badges, icons, and status indicators
- Clearly differentiate completed vs pending tasks
- Add micro-interactions for task completion toggles

Skill 4: Responsive & Mobile-First Design
- Ensure layouts adapt to mobile, tablet, desktop
- Improve touch targets and readability on small screens
- Avoid overflow and layout breaking on resize

Skill 5: Tailwind Styling Consistency
- Normalize colors, typography, border radius, shadows
- Prefer Tailwind utilities over custom CSS
- Use reusable style patterns across components

Skill 6: Accessibility & UX Basics
- Ensure proper focus styles
- Maintain contrast and readable font sizes
- Improve form usability and error visibility

Constraints (Global):
- UI-only changes
- No business logic, routing, or state flow changes
- No backend, auth, or database changes
- Lightweight enhancements only (no heavy UI libraries)
